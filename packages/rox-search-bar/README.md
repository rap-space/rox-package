# rox-search-bar

搜索框

---


### SearchBar

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
value | 默认值  | string | 无
placeholder | placeholder | string | 无
menuIndex | 菜单默认选中的项 | number | 0
menuData | 菜单数组 | array | 无
onClickMenu | 菜单点击事件 | function(index) | 无
onChange | 搜索框change事件 | function(value,e)  | 无
